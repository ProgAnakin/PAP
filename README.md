# PAP — Mobile App
### School Final Project | Built at 18 | Flutter/Dart

---

## What is this?

A mobile application built as my technical school final project
at Escola Secundária Martins Sarmento, Portugal (2023).

The entire UI/UX was designed from scratch in Figma before development began —
every screen, every flow, every interaction mapped before a single line of code.

Built with Flutter/Dart and a NoSQL database.

---

## What I learned

This was my first complete product — from concept to design to code to submission.

At 18, I learned:
- How to think in screens and user flows before writing code
- The discipline of designing in Figma before building
- How Flutter handles state and navigation
- What it feels like to ship something that is entirely yours

---

## Tech Stack

- **Framework:** Flutter / Dart
- **Design:** Figma (full UI/UX from scratch)
- **Database:** NoSQL
- **Platforms:** Android / iOS
